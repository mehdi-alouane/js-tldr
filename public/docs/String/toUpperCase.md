# String.prototype.toUpperCase()

The `toUpperCase` method returns the `string` converted to uppercase.

## Syntax

```js
const lowerString = string.toUpperCase();
```

## Usage examples

```js
'alphabet'.toUpperCase(); // -> 'ALPHABET'
```
